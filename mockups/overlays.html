The four overlay primitives committed in DESIGN.md § Overlays.

Modal — destructive confirmation

Delete Fairview Health?

This will permanently remove Fairview Health, all its contacts, projects, notes, and agent threads. This cannot be undone.
Esc to cancel

Trigger: "Delete" action on org settings page.   Esc: closes dialog, returns focus to the trigger element.   Focus-trap: Tab cycles between Cancel and Delete Customer only while modal is open. Delete button has autofocus and a vermilion border per Q-1 (not filled — filled vermilion fails AA at body size).

Toast — top-right anchored, aria-live="polite"
API key saved.
Settings · just now
Agent insight saved to Fairview Health.
Awaiting your review in Insights

Trigger: any successful mutation (save, create, delete).   Esc / dismiss (×): removes the toast immediately. Auto-dismisses after 6 s. Width 280px; stacks top-right (position:fixed; top:16px; right:16px). Confirm variant carries --accent left border; info uses --ink-3. aria-live="polite" on the container announces to screen readers.

Popover — anchored to Quick Reference button
Industry Health System
Region Midwest

Trigger: click the Profile button in the Quick Reference tile.   Esc: closes popover, returns focus to trigger.   8px border-radius; hairline --rule border; --bg-2 background; no drop shadow. Trigger gains border-color: var(--accent) while open (transient vermilion).
